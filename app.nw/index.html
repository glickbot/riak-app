Riak

Thank you for installing Riak! Below you'll find a summary of the functionality included in this dashboard.

Riak Control

 
 

Riak Control is a web-based administrative console for inspecting and manipulating Riak clusters.

Click the "Riak Control" button on the left to show / hide the iframe. Alternatively you can directly visit http://localhost:8098/admin to use this interface.

More information about Riak Control can be found at http://docs.basho.com/riak/latest/ops/advanced/riak-control

View Logs

 
File: etc/console.log:
 

Click the "View Logs" button on the left to show / hide the last 20 lines of log/console.log. Alternatively you can visit the log directory directly at ../__RIAK_DIR__/log.

Documentation

If you're new to Riak, don't hesitate to visit "A taste of Riak" at http://docs.basho.com/riak/latest/dev/taste-of-riak.

More useful information about Riak can be found at http://docs.basho.com/riak/latest

Actions

Use the Riak Start / Stop buttons on the left to perform those actions on this Riak node. To exit this application, use the Riak icon in your upper task bar to select "quit". Doing this will automatically stop the running Riak instance.
